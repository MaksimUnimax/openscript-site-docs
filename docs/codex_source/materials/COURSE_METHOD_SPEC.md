# Course Method Spec

Course inside materials:

**Как разрабатывать с помощью ChatGPT и Codex**

This course supports the main product.

It teaches the method:
- how a beginner works with ChatGPT and Codex;
- why documents matter;
- why context must be restored after a break;
- why roadmap matters;
- why git/history matters in simple terms;
- why one step at a time is safer;
- how GPT explains Codex reports;
- how the user checks result visually.

The course does not teach:
- Python;
- FastAPI;
- databases;
- manual Codex prompt writing;
- full app development inside the course.

Exercises:
tests only, with ready answers.

## Course method update 2026-05-28 — Student-facing method

The course inside “Работа с ИИ” teaches the working method, not programming syntax.

Current required principles:

- Course is inside the main site materials area.
- Course is not the root product.
- User should not be forced to write free-form Codex prompts.
- GPT prepares Codex prompts.
- User brings Codex report back to GPT.
- GPT explains the report in simple language.
- Exercises must be tests with ready answers.
- Course must be visual, not text-only.
- Lesson 7 includes a simple Git/GitHub/deploy-key explanation for students.
- Deploy key explanation must stay student-facing unless a future explicit decision changes that.
