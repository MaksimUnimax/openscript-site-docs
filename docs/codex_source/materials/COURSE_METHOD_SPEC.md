# Course Method Spec

Course inside materials:

**Как разрабатывать с помощью ChatGPT и Codex**

This course supports the main product.

It teaches the method:
- how a beginner works with ChatGPT and Codex;
- why documents matter;
- why context must be restored after a break;
- why roadmap matters;
- why git/history matters in simple terms;
- why one step at a time is safer;
- how GPT explains Codex reports;
- how the user checks result visually.

The course does not teach:
- Python;
- FastAPI;
- databases;
- manual Codex prompt writing;
- full app development inside the course.

Exercises:
tests only, with ready answers.
