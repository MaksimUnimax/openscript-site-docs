# Project snapshot directory

STATUS: INITIAL_FROM_CURRENT_REPO_DOCS

This directory stores the generated lightweight repo snapshot and its manifest.

Use it when:
- you need a quick tree-level view of the repo
- you need a bounded summary of directories without reading large files

Do not use it as:
- a substitute for the full project docs
- a substitute for vendor docs
- a substitute for append-only memory tails
