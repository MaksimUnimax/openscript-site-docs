# Project Overview

## Name

OpenScript / AI Starter Community

## Main site

https://openscript.ru

## Main app repo

`/opt/ai-starter-community`

## Product idea

OpenScript is a website and personal cabinet for people without programming experience who want to start making simple programs, bots, scripts, automations and MVPs with AI tools.

The product is not a classic programming course.
It is not a Python course.
It is not development “done for you”.

It is a guided starting environment:
- public landing;
- registration;
- personal cabinet;
- paid access model;
- materials section;
- courses inside the materials section;
- paid environment options;
- admin management;
- future payments;
- future AI-sales agent.

## Audience

Russian-speaking beginners who:
- do not know programming;
- want to try making useful tools;
- have ideas but no budget for developers;
- run small business/self-employment;
- need bots, scripts, parsing, tables or automations;
- need a clear process, not abstract AI hype.

## Core promise

A person can start making simple useful tools without knowing code at the beginning.

They do not need to become a professional developer in one month.
They do need:
- working environment;
- clear instructions;
- materials;
- ChatGPT + Codex workflow;
- community/support;
- safe step-by-step process.
