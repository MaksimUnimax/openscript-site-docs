# Paid options activation block

STATUS: CURRENT
PROJECT: OpenScript / AI Starter Community

## Purpose

The cabinet now includes an `Активация опций` block that shows active add-ons available from the paid-options catalog.

## Current live behavior

- Active add-ons are read from the database and rendered in `/cabinet`.
- The base AI / GPT tool option is hidden from the add-on activation block.
- Displayed add-ons are ordered by price descending.
- The buy button is currently a safe placeholder.
- Real payment processing is still NOT_YET_PROVEN.
