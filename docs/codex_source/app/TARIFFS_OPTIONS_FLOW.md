# Tariffs and Paid Options Flow

## Main model

Tariff:
- package user buys.

Paid option:
- separate paid component included in a tariff or extended separately.

Initial options:
- AI tool / GPT;
- server;
- VPN.

Future options:
- mentor;
- extra server;
- setup help;
- project review;
- closed workshop;
- other services.

## Important rule

Options must not be hardcoded.
Admin must be able to manage them.
