# Materials Flow

## Purpose

“Работа с ИИ” is the paid materials area.

It should contain:
- starter videos;
- step-by-step instructions;
- copyable commands;
- rules for working with AI agents;
- prompt templates;
- common errors and solutions;
- community/chat links;
- course materials;
- environment status;
- extension/payment information.

## Access rule

First successful payment grants permanent materials access.

## Course placement

Courses live here, inside the main product, after payment.
