# Homepage Spec

The homepage must explain OpenScript simply.

Core message:
People without programming experience can start making simple programs, bots and automations with AI tools, clear materials and guided workflow.

Avoid:
- heavy AI/IT jargon;
- unrealistic promises;
- “become developer in one month”;
- guaranteed income/job claims;
- “we build everything for you”.

Homepage must connect to:
- registration;
- cabinet;
- payment/access model;
- materials after payment.
