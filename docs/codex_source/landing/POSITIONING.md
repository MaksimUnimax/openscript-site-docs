# Positioning

OpenScript is not a classic programming course.

OpenScript is a starting environment and method for people who want to begin making useful tools with AI.

Simple positioning:

“Начните делать программы без знания кода.”

Expanded:

“Получите рабочую среду, понятные материалы и метод, с которым сможете собирать простые боты, скрипты и автоматизации под свои задачи.”
