# LiaScript

Source URLs:
- official site: https://liascript.github.io/
- upstream org: https://github.com/LiaScript/LiaScript

Date checked:
- 2026-05-28

What it does:
- provides an open Markdown dialect and browser interpreter for interactive courses
- supports quizzes, self-checks, animations, multimedia, live coding, SCORM export, and static hosting

Why it matters for OpenScript:
- useful as a fallback or reference when we want a Markdown-first course format
- fits the idea of course content living in versioned source, not runtime-only state

Setup/install notes:
- not performed in this docs run
- the public site shows browser-based authoring and optional tools such as LiveEditor, VSCode preview, and development server workflows
- no local installation was required to read the reference docs

Required secret categories:
- none for basic authoring and reading
- optional GitHub account if you want to host or export via GitHub-based workflows

Current status:
- `imported_reference`

Workflow fit:
- fallback/reference course format
- good for interactive Markdown-first prototypes and lightweight publishing

What not to do yet:
- do not use this to replace the first course design workflow without comparing it to the other tools
- do not treat it as an installed dependency in this repo
