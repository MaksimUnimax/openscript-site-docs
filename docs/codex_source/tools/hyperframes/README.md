# HyperFrames

Source URLs:
- upstream repo: https://github.com/heygen-com/hyperframes
- raw README: https://raw.githubusercontent.com/heygen-com/hyperframes/main/README.md

Date checked:
- 2026-05-28

What it does:
- turns HTML, CSS, media, and seekable animation into deterministic MP4 video
- supports local CLI preview/render workflows
- has skills for agent-driven video production

Why it matters for OpenScript:
- useful later for short visual snippets, animated explainers, and course support media
- good reference for making the course visually richer without turning the first shell into a video project

Setup/install notes:
- not performed in this docs run
- upstream docs require Node.js 22+ and FFmpeg
- upstream examples use `npx hyperframes init`, `preview`, and `render`

Required secret categories:
- none documented for the local CLI reference

Current status:
- `imported_reference`

Workflow fit:
- later visual/video layer
- not the first course storage or authoring path

What not to do yet:
- do not depend on video generation for the first usable course shell
- do not install or render anything in this docs-only run
